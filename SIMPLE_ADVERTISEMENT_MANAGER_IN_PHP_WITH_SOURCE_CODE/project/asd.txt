WYSIWYG Editor PHP MySql Javascript Source Code

This a simple application that will teach 
you on how to create a WSIYWYG rich text html
editor using PHP MySql and javascript.
This WYSIWYG Editor have multiple features
like changing font style,color,alignment.

Instructions:
1. Download the source code.
2. Extract the compressed folder.
3. Run the xampp control panel and start the apache and mysql
4. Copy the extracted folder to htdocs
5. Open the browser and type localhost/phpmyadmin
6. import the database wysiwyg
7. Run the program by typing in any browser localhost/script
